Architectures API

List architectures

GET /api/v1/arches.json

Response:

{
  "architectures": [
    {
      "id": "architecture identifier",
      "name": "architecture name"
    }
  ]
}

Response example:

{
  "architectures": [
    {
      "id": 1,
      "name": "x86_64"
    },
    {
      "id": 2,
      "name": "i586"
    }
  ]
}
